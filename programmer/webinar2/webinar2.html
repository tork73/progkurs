Litt om lister og arrayer i Python#

Lister i Python er en type variabel som kan inneholde flere elementer. Elementene i en liste kan være av ulik type. For eksempel kan en liste inneholde både tall og tekst.

liste = [1, 2, 3, 4, 5]

Arrayer i Python er en type variabel som kan inneholde flere elementer. Elementene i en array må være av samme type. For eksempel kan en array inneholde kun tall.

import numpy as np
array = np.array([1, 2, 3, 4, 5])

Kode

Forklaring

L[i]

Plukker ut element nr i

L[-1]

Det siste elementet i L

L[i:j]

Elementene med indeks fra og med i til j

L[i::j]

Elementene med indeks fra og med indeks i, der indeksen øker med j til indeksen er høyere enn antall element i listen

L.append(a)

Legger a til i listen som siste element

len(L)

Antall elementer i L

L.sort()

Sorterer liste i stigende rekkefølge

L.sort(reverse=True)

Sorterer liste i synkende rekkefølge

L.pop()

Fjerner siste element fra listen

L.pop(i)

Fjerner element med indeks i fra listen

L.remove(a)

Fjerner første forekomst av a fra listen

del L[i]

Fjerner element med indeks i fra listen.

L.reverse()

Snur rekkefølgen på elementene i listen

max(L)

Gir største tall i listen

min(L)

Gir minste tall i listen

sum(L)

Summerer elementene i listen

De viktiste funksjonene for arrayer er:

Funkjon

Beskrivelse

np.array(L)

Oppretter en array fra en liste L

np.arange(a, b, c)

Oppretter en array med tall fra a til b med steglengde c

np.linspace(a, b, n)

Oppretter en array med n tall fra a til b

np.zeros(n)

Oppretter en array med n nuller

np.ones(n)

Oppretter en array med n enere

np.random.rand(n)

Oppretter en array med n tilfeldige tall mellom 0 og 1

np.random.randint(a, b, n)

Oppretter en array med n tilfeldige heltall fra a til b

np.max(A)

Returnerer den største verdien i A

np.min(A)

Returnerer den minste verdien i A

np.sum(A)

Returnerer summen av elementene i A

np.sort(A)

Sorterer A i stigende rekkefølge
